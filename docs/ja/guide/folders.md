# フォルダ、あるべき姿。

AI のチャット履歴整理、以前はなぜあんなに大変だったのでしょう？
私たちが直しました。あなたの思考に、ファイルシステムを装備しましょう。

## 整理の「手触り」

手触りが良ければ、すべてが上手くいきます。

- **ドラッグ＆ドロップ**：掴んで、放り込む。リアルな物理的フィードバック。
- **入れ子構造**：大きなプロジェクトの中に小さなプロジェクトを。無限の階層構造もお好みで。
- **同期**：デスクトップで整理すれば、ノートパソコンでもすぐに使えます。

## 便利な機能

- **複数選択**：チャット項目を長押しすると複数選択モードに入り、一括操作でまとめて処理できます。
- **名前変更**：フォルダをダブルクリックするだけで、直接リネームできます。
- **アイコン**: Gem のタイプ (コーディング、クリエイティブなど) を自動的に検出し、適切なアイコンを割り当てます。何もする必要はありません。
- **配色**: フォルダの横にある3つのドットをクリックして、8色の鮮やかな色から好きなものを選べます。サイドバーをあなたの思考と同じくらいカラフルに。
