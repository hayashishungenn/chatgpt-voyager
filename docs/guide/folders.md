# 文件夹，本该如此

整理 AI 聊天记录，以前怎么那么难？
我们修好了。给你的思绪，装个文件系统。

## 整理的直觉

手感对了，一切都对了。

- **拖拽**：抓起来，扔进去。真实物理反馈。
- **套娃**：大项目套小项目。无限层级，随你怎么。
- **同步**：电脑上理好，笔记本上就能用。

## 绝招

- **多选**：长按对话项进入多选模式，批量操作，一次搞定。
- **改名**：双击文件夹，直接改。
- **识图**：代码、写作、闲聊... 我们自动识别 Gem 类型，配上图标。你只管用，剩下的交给我们。
- **配色**：点击文件夹侧边的三个点自定义颜色，8 种亮丽色彩任你选。让侧边栏像你的思绪一样缤纷。
