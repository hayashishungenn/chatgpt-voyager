# 資料夾，本該如此

整理 AI 聊天記錄，以前怎麼那麼難？
我們修好了。給你的思緒，裝個文件系統。

## 整理的直覺

手感對了，一切都對了。

- **拖拽**：抓起來，扔進去。真實物理回饋。
- **套娃**：大項目套小項目。無限層級，隨你怎麼。
- **同步**：電腦上理好，筆記本上就能用。

## 絕招

- **多選**：長按對話項進入多選模式，批次操作，一次搞定。
- **改名**：雙擊資料夾，直接改。
- **識圖**：代碼、寫作、閒聊... 我們自動識別 Gem 類型，配上圖標。你只管用，剩下的交給我們。
- **配色**：點擊資料夾側邊的三個點自定義顏色，8 種亮麗色彩任你選。讓側邊欄像你的思緒一樣繽紛。
