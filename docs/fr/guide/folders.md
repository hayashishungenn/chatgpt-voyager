# Les Dossiers, Bien Faits

Pourquoi organiser les chats IA est-il si difficile ?
Nous avons réglé le problème. Nous avons construit un système de fichiers pour vos pensées.

## La Physique de l'Organisation

C'est juste naturel.

- **Glisser-Déposer** : Prenez un chat. Déposez-le dans un dossier. C'est tactile.
- **Hiérarchie Imbriquée** : Les projets ont des sous-projets. Créez des dossiers dans des dossiers. Structurez-le à _votre_ façon.
- **Synchro Instantanée** : Organisez sur votre bureau. Retrouvez-le sur votre portable.

## Conseils de Pro

- **Multi-Sélection** : Appui long sur une conversation pour entrer en mode multi-sélection, puis sélectionnez plusieurs chats et déplacez-les tous en une fois.
- **Renommer** : Double-cliquez sur n'importe quel dossier pour le renommer.
- **Icônes** : Nous détectons automatiquement le type de Gem (Codage, Créatif, etc.) et attribuons la bonne icône. Vous n'avez rien à faire.
- **Couleurs** : Cliquez sur les trois points à côté d'un dossier pour choisir parmi 8 couleurs vibrantes. Rendez votre barre latérale aussi unique que vos pensées.
