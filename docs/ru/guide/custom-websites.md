# Пользовательские сайты для Менеджера промптов

Менеджер промптов теперь можно использовать на любом веб-сайте по вашему выбору, а не только в Gemini и AI Studio.

## Как это работает

Когда вы добавляете пользовательский сайт:

1. Расширение запрашивает разрешение на доступ только к этому конкретному веб-сайту
2. Менеджер промптов внедряется на сайт
3. **Только** функция Менеджера промптов активируется (без таймлайна, папок и других функций)

## Добавление веб-сайта

1. Нажмите на значок расширения, чтобы открыть всплывающее окно
2. Прокрутите до раздела **Менеджер промптов**
3. Введите URL веб-сайта (например, `chatgpt.com` или `claude.ai`)
4. Нажмите **Добавить сайт**
5. Предоставьте разрешение при появлении запроса

## Поддерживаемые форматы URL

Вы можете вводить URL в различных форматах:

- `chatgpt.com`
- `www.chatgpt.com`
- `https://chatgpt.com`
- `claude.ai`

Расширение автоматически нормализует URL и запрашивает разрешение как для `https://`, так и для `https://www.` версий.

## Конфиденциальность и безопасность

- **Контроль пользователя**: Вы явно добавляете веб-сайты, где хотите использовать Менеджер промптов
- **Ограниченная функциональность**: На пользовательских сайтах активируется только Менеджер промптов (без таймлайна, папок и т.д.)
- **Простое управление**: Добавляйте или удаляйте веб-сайты в любое время во всплывающем окне расширения

## Примеры

Популярные сайты ИИ-чатов, где вы, возможно, захотите использовать Менеджер промптов:

- `chatgpt.com` - ChatGPT
- `claude.ai` - Claude
- `copilot.microsoft.com` - Microsoft Copilot
- `bard.google.com` - Google Bard (устаревший)
- `poe.com` - Poe

## Удаление веб-сайта

1. Откройте всплывающее окно расширения
2. Найдите веб-сайт в списке
3. Нажмите **Удалить**
4. Контент-скрипт будет автоматически отменен

## Устранение неполадок

### Менеджер промптов не появляется после добавления веб-сайта

1. **Перезагрузите страницу** - Расширение активируется только при загрузке страницы
2. Проверьте правильность формата URL (например, `chatgpt.com`, а не `chat.openai.com/chatgpt`)
3. Откройте консоль браузера (F12) и ищите логи `[Gemini Voyager]`

### Как проверить, работает ли это

Откройте консоль браузера (F12) и ищите:

```
[Gemini Voyager] Checking custom websites: ...
[Gemini Voyager] Is custom website: true
[Gemini Voyager] Custom website detected, starting Prompt Manager only
```

### Хотите использовать другие функции на пользовательских сайтах?

В настоящее время на пользовательских сайтах поддерживается только Менеджер промптов. Другие функции (Таймлайн, Папки и т.д.) специально разработаны для Gemini и AI Studio и не будут работать на других сайтах.
